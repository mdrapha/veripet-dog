# Results

This folder contains small tabular artifacts used to document the reported VeriPet Dog metrics.

Included:

- `classification/metricas_por_raca_stanford.csv`
- `classification/top10_piores_racas.csv`
- `classification/distribuicao_erros_top10.csv`
- `classification/erros_top10_racas_qualitativo.csv`
- `classification/label_encoder_stanford.json`
- `verification/verification_results.csv`

Not included:

- raw predictions over full image datasets;
- image files;
- model checkpoints;
- extracted dataset archives.
